# 🎥 Ytdown - Multi-Platform Downloader

Developed by **SSK x Gemini**. This repository contains professional YouTube downloading tools for both Android (Termux) and Windows.

## 📂 Select Your Platform
* **[📱 Termux Version](./Termux)** - Best for Android users.
* **[💻 Windows Version](./Windows)** - Best for PC users.

---

## 🚀 Quick Start
- **Android:** Run the one-line command found in the Termux folder.
- **Windows:** Download `ytdown.bat` from the Windows folder and run it.

## ✨ Core Features
- High-quality MP4/MP3 downloads.
- Full Playlist & Range support.
- Cookie support for age-restricted videos.
- Auto-dependency installation.

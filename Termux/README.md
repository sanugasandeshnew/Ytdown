# 🎥 Ytdown - Professional YouTube Downloader

A powerful, user-friendly, and adaptive YouTube Downloader specifically designed for **Termux**. Developed by **SSK** in collaboration with **Gemini**.

## ✨ Key Features
- **Adaptive UI:** Automatically adjusts to any terminal zoom level.
- **Smart Validation:** Validates URLs and prevents empty inputs.
- **High Quality:** Supports Best MP4, 720p, and High-Quality MP3.
- **Playlist Pro:** Download entire playlists with "New Only" or "Range" options.
- **Cookie Manager:** Dedicated section to handle cookies for private/restricted videos.
- **Auto-Update:** Automatically checks and updates dependencies on startup.

## 🚀 One-Line Installation
Copy and paste this command in your Termux:
```bash
curl -sL [https://raw.githubusercontent.com/sanugasandeshnew/Ytdown/main/tool.sh](https://raw.githubusercontent.com/sanugasandeshnew/Ytdown/main/tool.sh) -o tool.sh && chmod +x tool.sh && ./tool.sh

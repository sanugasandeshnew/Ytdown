## 🛠️ භාවිතා කරන ආකාරය (How to Use)

පරිගණක තාක්ෂණය ගැන දැනුමක් නැති කෙනෙකුට වුණත් ඉතා පහසුවෙන් මෙය භාවිතා කළ හැක:

1.  **බාගත කරන්න (Download):** මෙම ෆෝල්ඩරයේ ඇති [`ytdown.bat`](./ytdown.bat) ෆයිල් එක බාගත කරගන්න.
2.  **ක්‍රියාත්මක කරන්න (Run):** `ytdown.bat` මත Double-Click කරන්න.
3.  **ස්වයංක්‍රීය සැකසුම (Auto Setup):** පළමු වරට පාවිච්චි කරන විට, මෘදුකාංගයට අවශ්‍ය සියලුම දේවල් (yt-dlp සහ FFmpeg) එය විසින්ම සොයාගෙන හෝ Install කරගනු ඇත.
4.  **බාගත කරන්න (Start Downloading):** වීඩියෝ එකේ ලින්ක් එක ලබා දී ඔබට අවශ්‍ය Format එක තෝරන්න.

# 💻 Ytdown for Windows

This is the professional Windows version of the YouTube Downloader.

### 🚀 How to Run:
1. Download the `ytdown.ps1` file from this folder.
2. Right-click on the file and select **"Run with PowerShell"**.
3. If it asks for permission, type `Y` and press **Enter**.

### 🛠️ Features:
- Automatically downloads `yt-dlp` and `ffmpeg` if missing.
- High-quality MP4 and MP3 downloads.
- Simple command-line interface.
